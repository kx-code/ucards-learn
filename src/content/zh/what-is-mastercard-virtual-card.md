## Mastercard（万事达卡）介绍

Mastercard（万事达卡）是全球第二大支付网络，成立于 1966 年，总部位于美国纽约州。与 Visa 类似，Mastercard 不直接发行卡片，而是运营连接全球金融机构的支付处理网络。

## 什么是 Mastercard 虚拟卡？

Mastercard 虚拟卡是没有实体塑料形态的数字化万事达支付卡。它拥有标准 Mastercard 卡号格式（以 5 开头的 16 位数字）、有效期和 CVV 安全码，在所有接受 Mastercard 的在线商户中均可使用。

### Mastercard 虚拟卡工作原理

```
用户注册 → 获取虚拟万事达卡号 → 加密货币充值 → 在线支付 → Mastercard网络处理
```

交易处理流程与实体 Mastercard 卡完全相同：

1. **输入卡信息** — 在商户结账页面输入卡号、有效期和 CVV
2. **安全验证** — Mastercard SecureCode（3D Secure）进行身份验证
3. **授权请求** — 通过 Mastercard 网络发送授权请求
4. **余额检查** — 验证卡内余额是否充足
5. **交易完成** — 授权通过，商户收到付款确认

### 商户视角

商户无法区分你使用的是虚拟卡还是实体卡——在他们看来，这就是一笔标准的 Mastercard 交易。你享受与实体 Mastercard 卡完全相同的支付体验和安全性。
