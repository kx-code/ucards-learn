## 德国的信用体系壁垒：Schufa 问题

Schufa（德国通用信用保障机构）是德国金融体系的核心。几乎每一家银行在审批信用卡时都会查询 Schufa 评分：

| Schufa 问题  | 影响                                               |
| ------------ | -------------------------------------------------- |
| 新移民无记录 | 无法通过信用卡审批，即使有稳定收入                 |
| 留学生身份   | 大多数银行拒绝向学生发放信用卡                     |
| 评分降低     | 一次逾期付款可能导致数年内无法获批                 |
| 查询记录影响 | 每次信用卡申请都会留下查询记录，频繁申请会降低评分 |
| 隐私担忧     | Schufa 收集了大量个人消费和信贷数据                |

据统计，德国约有 **800 万人**因 Schufa 评分问题而难以获取传统信用卡。这其中包括数百万在德国工作的欧盟及非欧盟外籍人士。
