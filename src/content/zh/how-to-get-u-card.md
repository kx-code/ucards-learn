## 前置准备

在开始注册 uCards 之前，你需要准备以下两样东西：

### 1. 一个加密钱包

你需要一个支持 USDT 的加密钱包。以下是推荐选项：

| 钱包         | 支持网络                 | 推荐场景           |
| ------------ | ------------------------ | ------------------ |
| TronLink     | TRON (TRC20)             | 最推荐，手续费最低 |
| MetaMask     | ETH (ERC20)、BSC (BEP20) | 通用型钱包         |
| Trust Wallet | TRON、ETH、BSC           | 移动端用户         |
| TokenPocket  | TRON、ETH、BSC           | 国内用户友好       |

如果你还没有钱包，TronLink 是最简单的选择。它有浏览器插件和手机 App，安装后创建钱包即可使用。

### 2. 一些 USDT

你的钱包中需要有足够的 USDT 来支付开卡费和充值金额。最低要求：

- 开卡费：$0 - $2.50（取决于卡类型）
- 首次充值：最低 $10
- 区块链 Gas 费：约 $0.5 - $2（TRC20 最便宜）

如果你还没有 USDT，可以在以下交易所购买：

- 币安（Binance）
- OKX
- Bybit
- HTX（火币）

购买后，将 USDT 提现到你的个人钱包地址（不要直接从交易所充值到 uCards，部分交易所可能限制此操作）。

## 第一步：访问 uCards 官网

打开浏览器，访问 [ucards.uk](https://ucards.uk)。

uCards 支持中英文双语界面。网站会自动根据你的浏览器语言设置显示对应版本。你也可以通过页面顶部的语言切换按钮手动切换。

## 第二步：注册账户

点击页面上的注册/登录按钮。uCards 提供两种注册方式：

### 方式一：Google 账号登录（推荐）

这是最快的方式：

1. 点击「使用 Google 登录」
2. 在弹出的 Google 登录窗口中选择你的 Google 账号
3. 授权后自动完成注册

整个过程不到 20 秒。

### 方式二：邮箱注册

1. 输入你的邮箱地址
2. 设置密码
3. 点击注册
4. 到邮箱中查收验证邮件并点击确认链接

邮箱注册大约需要 1-2 分钟。

### 注册注意事项

- 不需要填写真实姓名
- 不需要上传身份证件
- 不需要填写地址或电话号码
- 建议使用常用邮箱，以便接收账户通知

## 第三步：选择卡类型

登录后，你会看到 uCards 提供的所有卡类型。根据你的使用需求选择合适的卡片：

### 如何选择？

| 你的需求       | 推荐卡片                        | 原因                        |
| -------------- | ------------------------------- | --------------------------- |
| 通用在线消费   | Visa 全能卡                     | Visa 接受度最广，无行业限制 |
| 需要最大接受度 | Visa 全能卡或 MasterCard 全能卡 | 两个卡组织互补覆盖          |
| 订阅 AI 服务   | Visa AI 卡                      | 专为 AI 订阅优化，开卡费低  |
| 广告投放       | Visa 广告卡                     | 免费，专为广告平台设计      |
| 预算有限       | Visa 广告卡或 Discover 全能卡   | 开卡费最低/免费             |

### 限额对比

不同卡类型有不同的使用限额：

| 卡片              | 日限额 | 月限额  | 单笔限额 |
| ----------------- | ------ | ------- | -------- |
| Visa 全能卡       | $5,000 | $15,000 | $1,000   |
| MasterCard 全能卡 | $3,000 | $10,000 | $500     |
| Discover 全能卡   | $2,000 | $6,000  | $400     |
| Visa AI 卡        | $3,000 | $10,000 | $500     |
| Visa 广告卡       | $5,000 | $15,000 | $1,000   |

选择好卡类型后，点击对应的「开通」或「购买」按钮。

## 第四步：用 USDT 支付

### 4.1 查看支付信息

选择卡类型后，系统会显示：

- **开卡费**：$0 - $2.50
- **充值金额**：你可以选择首次充值金额（最低 $10）
- **总金额**：开卡费 + 充值金额 + 充值手续费（2%）
- **收款地址**：一个专属的 USDT 收款地址
- **支持的网络**：TRC20 / ERC20 / BEP20

### 4.2 选择支付网络

页面会让你选择使用哪条区块链网络发送 USDT。强烈推荐选择 **TRON (TRC20)**：

- 手续费不到 $1
- 确认速度最快（1-3 分钟）
- 全球 USDT 使用率最高的网络

### 4.3 发送 USDT

打开你的加密钱包，执行以下操作：

1. **复制收款地址**：从 uCards 页面复制显示的 USDT 收款地址
2. **打开钱包的发送功能**：在 TronLink 或其他钱包中点击「发送」
3. **粘贴地址**：将收款地址粘贴到收款人字段
4. **输入金额**：输入系统显示的总金额
5. **确认网络**：确保选择的是 TRC20 网络（极其重要！）
6. **确认并发送**：核对信息无误后确认交易

### 支付注意事项

- **网络必须匹配**：TRC20 地址发送 TRC20 USDT，ERC20 地址发送 ERC20 USDT。发错网络会导致资金丢失。
- **金额必须准确**：按系统显示的总金额发送，包含尾数。uCards 使用精确金额匹配来确认你的支付。
- **不要从交易所直接发送**：部分交易所可能不支持直接向个人地址转账，或可能延迟处理。建议先提现到个人钱包，再从钱包发送。
- **保留交易哈希**：如果支付出现问题，交易哈希（TxID）是追踪问题的重要凭证。

## 第五步：等待确认并获取卡信息

### 5.1 监控支付状态

支付发送后，uCards 页面会自动显示支付状态：

| 状态     | 含义                           |
| -------- | ------------------------------ |
| 等待支付 | 等待区块链上检测到交易         |
| 确认中   | 交易已被检测到，等待区块链确认 |
| 已完成   | 确认完成，卡片已激活           |

TRC20 网络通常 1-3 分钟完成确认。你可以在此期间保持页面打开，系统会自动刷新状态。

### 5.2 获取卡信息

支付完成后，进入「我的卡片」页面，你会看到：

- **卡号**（16 位数字）
- **有效期**（月/年）
- **CVV**（3 位安全码）
- **卡片状态**（活跃/未激活）
- **当前余额**

这些信息可以立即用于在线消费。

## 第六步：开始使用你的 U卡

### 在线消费

在任何在线商家的支付页面：

1. 选择信用卡支付
2. 输入卡号、有效期和 CVV
3. 账单地址可以使用任意美国地址（部分平台提供默认地址）
4. 完成支付

### 充值更多余额

当你需要更多余额时：

1. 进入「我的卡片」页面
2. 选择要充值的卡片
3. 点击「充值」
4. 输入充值金额
5. 用 USDT 支付到新的收款地址
6. 等待确认，余额即时更新

充值手续费为 2%，最低充值 $10。

## 常见问题与故障排除

### 支付发送了但状态一直是「等待支付」

可能原因和解决方法：

- **网络选择错误**：检查你是否发送到了正确的网络地址
- **金额不匹配**：确认发送金额与系统要求一致，包含尾数
- **区块链拥堵**：等待更长时间（特别是 ERC20 网络）
- **交易未广播**：检查钱包中的交易状态

如果超过 30 分钟仍未确认，记录下交易哈希并联系客服。

### 卡片支付被拒绝

- **余额不足**：确保卡内余额足以支付消费金额
- **商家风控**：部分商家对虚拟卡有额外验证，尝试先做一笔小额交易
- **超出限额**：检查是否超出了日限额或单笔限额
- **地区限制**：部分卡类型在特定地区不可用（如 CosmicPay 卡不适用于中国大陆）

### 如何查看交易记录

在「我的卡片」页面，选择对应卡片，点击「交易记录」即可查看所有消费明细。

### 可以开多张卡吗？

是的，你可以同时持有多张不同类型的卡片。每张卡有独立的余额和卡信息。

## 新手建议

### 资金管理

- **少量多次充值**：建议每次充值不超过你近期需要的金额，避免大额余额闲置
- **TRC20 优先**：始终优先使用 TRC20 网络充值，手续费最低
- **保留余量**：充值时多充一点（考虑 2% 手续费），避免余额恰好不够

### 安全建议

- **妥善保管登录信息**：由于没有 KYC，账户找回可能需要更多信息
- **不要分享卡信息**：卡号、有效期和 CVV 与实体信用卡一样重要
- **定期检查交易记录**：及时发现异常交易

### 消费建议

- **首次绑定先小额**：在新商家首次使用时，先做一笔 $1-2 的小额交易验证卡片可用
- **注意退款政策**：部分卡类型不支持退款，消费前了解商家的退款政策
- **广告卡专卡专用**：Visa 广告卡专为广告平台设计，日常消费建议使用全能卡
