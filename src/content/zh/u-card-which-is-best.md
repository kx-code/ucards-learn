**U卡**是用USDT（Tether）等加密货币充值的虚拟信用卡的俗称。因为USDT在中文社区中常被称为"U"或"泰达币"，所以用USDT充值的虚拟卡就被简称为"U卡"。

### U卡的工作原理

```
加密货币钱包 → USDT充值 → U卡平台转换 → Visa/Mastercard余额 → 在线消费
```

1. 你在加密货币交易所或钱包中持有USDT
2. 将USDT通过TRON/Ethereum/BSC网络转到U卡平台
3. 平台将USDT按1:1比例转换为美元余额
4. 你获得一张Visa/Mastercard虚拟卡
5. 在任何接受Visa/Mastercard的在线商家消费
