## What Is a Visa Virtual Card?

A Visa virtual card is a fully digital Visa payment card with no physical form. It has all the same payment capabilities as a traditional Visa card — a 16-digit card number, expiry date, and CVV security code — it just doesn't exist on a piece of plastic.

### How It Works

```
Register → Receive virtual card number (16 digits + expiry + CVV) → Add funds → Pay online → Visa network processes the transaction
```

When you pay online with a Visa virtual card, the transaction flow is identical to using a physical Visa card:

1. **Enter card details** — Input the 16-digit card number, expiry date, and CVV at the merchant's checkout page
2. **Authorization request** — The merchant sends an authorization request to the Visa network via a payment gateway
3. **Verification** — The Visa network verifies that the card number is valid and sufficient funds are available
4. **Transaction complete** — Authorization is approved and the transaction is finalized

Merchants and payment gateways cannot tell the difference between a virtual card and a physical one — they are both standard Visa transactions.
