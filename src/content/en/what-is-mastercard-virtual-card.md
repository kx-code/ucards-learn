## What Is a Mastercard Virtual Card?

A Mastercard virtual card is a fully digital Mastercard payment card with no physical plastic form. It features a standard Mastercard card number format (16 digits starting with 5), an expiry date, and a CVV security code, and works at all online merchants that accept Mastercard.

### How Mastercard Virtual Cards Work

```
Register → Receive virtual Mastercard number → Fund with cryptocurrency → Pay online → Mastercard network processes the transaction
```

The transaction flow is identical to using a physical Mastercard:

1. **Enter card details** — Input the card number, expiry date, and CVV at checkout
2. **Security verification** — Mastercard SecureCode (3D Secure) authenticates the transaction
3. **Authorization request** — Sent through the Mastercard network
4. **Balance check** — Verifies sufficient funds are available
5. **Transaction complete** — Authorization approved, merchant receives payment confirmation

### The Merchant's Perspective

Merchants cannot tell whether you're using a virtual card or a physical one — to them, it's a standard Mastercard transaction. You get the same payment experience and security as a physical Mastercard.
