### What Is KYC?

KYC (Know Your Customer) is the process financial institutions use to verify user identity. A typical KYC process includes:

| KYC Level        | Requirements                   | Common Platforms          |
| ---------------- | ------------------------------ | ------------------------- |
| **No KYC**       | Email or wallet only           | uCards, OneKey            |
| **Basic KYC**    | Email + phone number           | RedotPay (partial), Pyypl |
| **Standard KYC** | ID + selfie                    | Dupay, PST.net            |
| **Full KYC**     | ID + selfie + proof of address | Revolut, Wise             |

### Are no-KYC virtual cards the same as anonymous cards?

Not exactly.
